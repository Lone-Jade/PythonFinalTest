#### 第 6 页：核心创新 — ScaleInv 架构（1min）

```
┌─────────────────────────────────────────────────┐
│  核心创新：Scale-Invariant 神经网络架构            │
│                                                  │
│  问题：普通 MLP 在不同规模实例上的特征分布差异大     │
│        → 泛化到未见规模时性能暴跌                   │
│                                                  │
│  ┌─────────────────────────────┐                │
│  │  Input [19 dims]             │                │
│  │      │                       │                │
│  │  ┌── LayerNorm(input) ──┐   │  ← 消除规模差异  │
│  │  │                      │   │                │
│  │  ▼                      ▼   │                │
│  │ State Encoder    Action Encoder│ ← 分离编码    │
│  │ (10 dims)        (19 dims)     │   防信息泄漏  │
│  │  │                      │   │                │
│  │  ▼                      ▼   │                │
│  │ V(s) ← 价值评估    A(s,a) ← 优势比较│          │
│  │                      │   │                │
│  │           Q = V + A - mean(A) │                │
│  └─────────────────────────────┘                │
│                                                  │
│  效果：                                            │
│  · DQN: Test/H 从 135% → 120% (-11pp)            │

│  · 泛化差距从 +12.8pp → +1.9pp (-85%)             │
└─────────────────────────────────────────────────┘
```
