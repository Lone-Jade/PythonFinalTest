### 1.3 DuelingPairScoringNetwork 架构

```
features (19 dims)
    ├── state_feats (10 dims: global+worker) → Value Net → V(s)  (1,)
    └── full_feats  (19 dims: all)          → Advantage Net → A(s,a) (n_actions,)

Q(s,a) = V(s) + A(s,a) - mean(A(s,:))
```

参数：Value Net 37,250 + Advantage Net 37,250 = **74,500**（vs 原 PairScoringNet 37,250）

## 七、分析

### 7.1 500轮的价值

- **PPO 大实例明显受益**：50x10x2 改善 38%，100x10x2 改善 35%
- **DQN 改善温和**：大实例 14-17% 改善
- **但 plateau 早**：DQN ep=150, PPO ep=1 后验证无提升
- **Curriculum 效果不明显**：Stage 过渡时无明显跳跃

### 7.2 Dueling DQN 评价

- DQN 在中型实例上保持优势（5:0 S-M规模）
- 但在大实例上不如 PPO（0:10 XL+规模）
- 相比 100ep 的 PairScoringNet + PER，大实例改善 14-17%（部分可能来自500轮而非 Dueling）

### 7.3 未解决问题

1. **早期 plateau**：DQN ep=150, PPO ep=1 即停止改善 → replay buffer 污染 / policy collapse
2. **验证崩溃**：ep=450 时 DQN 在 6x6x3 上 makespan=5535, decisions=711
3. **大实例泛化**：50x10+ 仍落后启发式 10-30%
4. **训练集缺少大实例**：最大训练实例仅 20x10x3 (200任务)，测试却有 100x10 (1000任务)

## 十、下一步建议

当前瓶颈分析：

| 瓶颈 | 证据 | 建议 |
|------|------|------|
| 训练集缺大实例 | 最大训练 20x10，测试有 100x10 | **增加 30x10, 50x10 到训练集** |
| Early plateau | ep=150 后无改善 | Periodic buffer reset / 更大 replay |
| PPO 崩溃 | val 偶尔爆炸 (667964) | PPO clip range 衰减 / grad clipping |
| 规模泛化差 | 100x10 落后 25%+ | 大实例入训 / representation learning |
